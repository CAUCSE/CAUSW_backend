# etc 도메인

위치: `app-main/src/main/java/net/causw/app/main/domain/etc`

## 책임

**위 7개 도메인 중 어느 곳에도 명확히 속하지 않는 운영 보조 기능** 을 모은 도메인입니다. 기능 플래그, 정책 텍스트, 외부 API 호출 관리 등이 포함됩니다.

## 서브 도메인

```
domain/etc/
├── flag/         # 기능 플래그
├── textfield/    # 정책 / 공지 텍스트
└── api/          # 외부 API 호출 관리
```

## flag — 기능 플래그

`flag/entity/` 하위: `Flag`

용도:
- 특정 기능을 켜고/끄는 플래그 (예: 점검 모드, 새 기능 단계적 노출)
- 런타임에서 변경 가능 — DB 기반 플래그

ErrorCode: 공통 `ErrorCode.FLAG_NOT_AVAILABLE` (global 모듈)

사용 흐름:
1. 관리자 API 로 플래그 on/off
2. 서비스 코드에서 플래그 상태 확인 후 분기

## textfield — 정책 / 공지 텍스트

`textfield/entity/` 하위: `TextField`

용도:
- 약관, 공지, 안내 문구 등 클라이언트가 자주 가져가는 정적 문구
- 코드 배포 없이 운영자가 변경

## api — 외부 API 호출 관리

용도:
- 외부 API 호출 통계 / 한도 / 로그 관리
- 외부 키 / 엔드포인트 등록 (운영자 도구)

## 외부 도메인 의존

| 의존 도메인 | 사용 |
|------------|------|
| (없음 / 운영 보조) | 다른 도메인에서 단순 조회용으로 의존 |

## 주의 사항

- **느슨하게 묶인 도메인** — 신규 기능이 여기 들어가야 한다고 느껴진다면, 정말 다른 도메인에 속하지 않는지 다시 검토
- 기능 플래그 변경은 운영 영향이 크므로 관리자 인가 (`@PreAuthorize("hasRole('ADMIN')")`) 필수
- TextField 는 사용자에게 그대로 노출되므로 XSS 방어 (저장 단계에서 sanitize 또는 응답에서 escape)
